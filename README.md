# Link-in-bio (ТД СХР)

## Редактирование контента

Все ссылки/тексты/изображения редактируются в `content.js`.

- **Кнопки/заголовки/тексты UI**: `APP_CONTENT.labels`
- **Фон/иконки/логотип**: `APP_CONTENT.images`
- **Основатели**: `APP_CONTENT.founders`
- **Почта/адрес**: `APP_CONTENT.commonContacts`
- **Юр.данные**: `APP_CONTENT.legal`
- **Соцсети компании**: `APP_CONTENT.socials` (есть `isEnabled`)

После изменений сохраните файл и обновите страницу.

## Структура файлов

- `index.html` — разметка и подключения
- `styles.css` — читаемые стили
- `styles.min.css` — минифицированные стили (используются страницей)
- `app.js` — читаемая логика
- `app.min.js` — минифицированная логика (используется страницей)
- `fonts.css` — локальные шрифты (генерируется)
- `assets/fonts/` — файлы шрифтов `.woff2` (скачиваются)

## Сборка (локальные шрифты + минификация)

Запуск в PowerShell из папки проекта:

```powershell
.\build.ps1
```

Что делает:
- скачивает шрифты в `assets/fonts/` и генерирует `fonts.css`
- генерирует `styles.min.css` и `app.min.js`
